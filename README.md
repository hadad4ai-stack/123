# نموذج محلي · Local LLM (PWA)

تطبيق ويب يعمل على الآيفون عبر المتصفح، **يشغّل نماذج الذكاء الاصطناعي محلياً بالكامل**
داخل المتصفح عبر WebGPU (مكتبة [WebLLM](https://github.com/mlc-ai/web-llm)) — بدون أي خادم،
ولا تُرسل بياناتك لأي جهة. قابل للتثبيت على الشاشة الرئيسية كتطبيق (PWA).

## المميزات

- 🧠 **نماذج محلية**: Qwen2.5 (0.5B / 1.5B)، Llama 3.2 (1B / 3B)، Gemma 2 2B.
- 🔧 **استدعاء الأدوات (Tool / Function Calling)**: آلة حاسبة، الوقت والتاريخ، رقم عشوائي، إحصائيات نص.
- 🎯 **المهارات (Skills)**: محادثة عامة، رياضيات، برمجة، كتابة — لكل مهارة دور وأدوات افتراضية.
- 📝 **تعليمات نظام مخصّصة**: اكتب System Prompt خاصاً بك، يُطبَّق قبل تشغيل النموذج ويلتزم به النموذج بأقصى درجة.
- 📲 **PWA**: قابل للتثبيت ويعمل دون إنترنت بعد أول تحميل.
- 🔊 نطق الردود (TTS) اختياري.

## التشغيل محلياً

يجب تقديم الملفات عبر خادم HTTP (وليس فتح الملف مباشرة) حتى يعمل Service Worker و WebGPU:

```bash
# أي خادم ثابت بسيط
python3 -m http.server 8000
# أو
npx serve .
```

ثم افتح `http://localhost:8000`.

## التشغيل على الآيفون

> WebGPU متاح في **Safari على iOS 17 وأحدث**. على بعض الإصدارات قد تحتاج لتفعيله من:
> `الإعدادات ← Safari ← متقدّم ← Feature Flags ← WebGPU`.

1. انشر الملفات على استضافة تدعم **HTTPS** (مثل GitHub Pages).
2. افتح الرابط في Safari على الآيفون.
3. من شريط المشاركة اختر **«إضافة إلى الشاشة الرئيسية»**.
4. افتح التطبيق من الأيقونة، ثم من ☰ اختر النموذج واضغط **«تحميل النموذج»**
   (أول تحميل يجلب ملفات النموذج ويخزّنها للاستخدام لاحقاً دون إنترنت).

### النشر على GitHub Pages

فعّل Pages للمستودع من الفرع المطلوب (المجلد الجذر `/`)، وسيُخدم التطبيق عبر HTTPS مباشرة.

## البنية

```
index.html              # الواجهة
css/styles.css          # التنسيق (واجهة عربية RTL)
js/llm.js               # غلاف WebLLM (تحميل النموذج + بث الردود + الأدوات)
js/tools.js             # تعريف الأدوات ومنفّذوها
js/skills.js            # المهارات وبناء تعليمات النظام
js/app.js               # منطق التطبيق وحلقة المحادثة
manifest.webmanifest    # بيانات الـ PWA
sw.js                   # Service Worker للعمل دون إنترنت
icons/                  # الأيقونات (PNG)
tools/generate-icons.js # مولّد الأيقونات (Node، بدون اعتماديات)
```

## الخصوصية

كل المعالجة تتم على جهازك. الاتصال الوحيد بالشبكة هو **تنزيل ملفات النموذج لأول مرة**
من مستودع WebLLM العام، وبعدها يعمل التطبيق دون إنترنت.
